# INSTALL Installation Process

# Update Pkg
```apt update```

# Upgrade Pkg
```apt upgrade```

# Install Git Pkg
```apt install git```

# Install Python Pkg
```apt install python```

# Clone GitHub Repository
```git clone https://github.com/JosefNoniyar/INSTALL```

# Go to INSTALL Directory
```cd INSTALL```

# Give Permission To Run The Tool
```chmod +x *```

# Install Requirement
```sh requirement.sh```

# Run Program
```python Install.py```
